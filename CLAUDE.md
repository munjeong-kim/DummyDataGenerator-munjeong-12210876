# DummyDataGenerator

## 개요
테스트용 JSON 더미 데이터를 생성하는 도구의 PoC(Proof of Concept) 프로젝트입니다.

## 핵심 컨셉
- 로컬에 저장된 JSON 파일을 **스키마/템플릿**으로 사용합니다.
- 템플릿 JSON의 필드 구조와 타입(문자열, 숫자, 불리언, 배열, 중첩 객체 등)을 분석하여,
  동일한 구조를 유지하면서 값만 랜덤하게 채운 더미 데이터를 생성합니다.
- 생성된 더미 데이터는 테스트 코드/테스트 환경에서 사용하는 것을 목표로 합니다.

## 현재 상태
- PoC 단계이며 아직 소스 코드는 작성되지 않았습니다.
- Python 3.13 기반 프로젝트로 세팅되어 있습니다 (`.idea`, `.venv`).

## 향후 논의/결정 필요 사항
- 템플릿 JSON의 타입 추론 규칙 (예: 필드명 패턴 기반 추론 여부 - `email`, `id`, `date` 등)
- 랜덤 값 생성에 사용할 라이브러리 (예: Faker 등) 채택 여부
- 배열/중첩 객체의 반복 개수 결정 방식
- 생성 결과 출력 방식 (파일로 저장 / 함수 반환값 등)
